### **توضیح درباره صفحه لودینگ و دایره چرخان (Loading Spinner) با تم آبی**

#### **هدف**
صفحه لودینگ و دایره چرخان (Loading Spinner) برای سایت Xraynama طراحی شده تا وقتی کاربر بین صفحات جابه‌جا می‌شه یا یه عملیات سنگین (مثل لود شدن یه فیلم، جستجو یا ورود به سیستم) در حال انجامه، بهش نشون بده که سایت در حال کاره و منتظر بمونه. این باعث می‌شه تجربه کاربری (UX) بهتر بشه، چون کاربر احساس نمی‌کنه سایت هنگ کرده یا مشکلی پیش اومده.

#### **طراحی و ظاهر**
- **پس‌زمینه صفحه لودینگ:**
  - یه لایه نیمه‌شفاف (Overlay) با رنگ مشکی و شفافیت 0.8 (Opacity: 0.8) که کل صفحه رو می‌پوشونه.
  - این لایه باعث می‌شه محتوای زیرین صفحه (مثلاً صفحه اصلی یا صفحه فیلم) کمی تار بشه و تمرکز کاربر روی دایره چرخان باشه.
- **دایره چرخان (Spinner):**
  - یه دایره چرخان ساده با تم آبی (رنگ اصلی: #0078D4، آبی ویندوزی).
  - اندازه دایره: 60px (قطر).
  - ضخامت خط دایره: 6px.
  - انیمیشن چرخش: دایره به صورت ساعت‌گرد می‌چرخه (360 درجه در 1.2 ثانیه، با انیمیشن linear و بی‌نهایت تکرار).
  - برای جذابیت بیشتر، یه سایه نرم (Box-Shadow) به رنگ #005BB5 (آبی تیره‌تر) به دایره اضافه شده.
- **متن لودینگ:**
  - زیر دایره چرخان، یه متن ساده: "در حال بارگذاری..." (فارسی) یا "Loading..." (انگلیسی).
  - فونت: IRANSans برای فارسی، Segoe UI برای انگلیسی.
  - اندازه فونت: 16px.
  - رنگ متن: سفید (#FFFFFF) با یه سایه نرم (#000000، Opacity 0.2) برای خوانایی بهتر.
- **موقعیت:**
  - دایره چرخان و متن دقیقاً وسط صفحه (به صورت افقی و عمودی) قرار می‌گیرن (با CSS: position: fixed; top: 50%; left: 50%; transform: translate(-50%, -50%)).
- **تم آبی:**
  - همه المان‌ها با تم آبی طراحی شدن: دایره چرخان (#0078D4)، سایه‌ها (#005BB5)، و پس‌زمینه نیمه‌شفاف که با تم دارک سایت همخونی داره.

#### **کاربرد**
- **بین صفحات:** وقتی کاربر از صفحه اصلی به صفحه فیلم می‌ره (مثلاً xraynama.com/movie/inception)، اگه لود شدن صفحه بیشتر از 0.5 ثانیه طول بکشه، این لودینگ نمایش داده می‌شه.
- **عملیات سنگین:** موقع جستجو (مثلاً جستجوی "اکشن")، لود شدن ویدیو، یا ارسال فرم (مثلاً ورود به سیستم)، دایره چرخان نشون داده می‌شه.
- **زمان نمایش:** لودینگ فقط تا وقتی عملیات تموم بشه نمایش داده می‌شه (حداقل 0.5 ثانیه برای جلوگیری از چشمک زدن).

#### **کد نمونه (CSS و HTML)**
برای اینکه این دایره چرخان رو تو سایتت پیاده‌سازی کنی، می‌تونی از کد زیر استفاده کنی:

```html
<!-- HTML -->
<div class="loading-overlay" id="loading">
    <div class="spinner"></div>
    <p class="loading-text">در حال بارگذاری...</p>
</div>
```

```css
/* CSS */
.loading-overlay {
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background: rgba(0, 0, 0, 0.8);
    display: flex;
    flex-direction: column;
    justify-content: center;
    align-items: center;
    z-index: 9999;
    display: none; /* پیش‌فرض مخفی است */
}

.spinner {
    width: 60px;
    height: 60px;
    border: 6px solid #E0E0E0;
    border-top: 6px solid #0078D4;
    border-radius: 50%;
    animation: spin 1.2s linear infinite;
    box-shadow: 0 0 10px rgba(0, 91, 181, 0.3);
}

@keyframes spin {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

.loading-text {
    color: #FFFFFF;
    font-family: 'IRANSans', sans-serif;
    font-size: 16px;
    margin-top: 10px;
    text-shadow: 0 0 5px rgba(0, 0, 0, 0.2);
}
```

```javascript
// JavaScript (برای نمایش/مخفی کردن لودینگ)
function showLoading() {
    document.getElementById('loading').style.display = 'flex';
}

function hideLoading() {
    document.getElementById('loading').style.display = 'none';
}

// مثال: موقع کلیک روی یه لینک
document.querySelector('a').addEventListener('click', function() {
    showLoading();
    // بعد از لود شدن صفحه، مخفی کن
    setTimeout(hideLoading, 2000); // فقط برای تست
});
```

#### **بهینه‌سازی‌ها**
- **سرعت:** دایره چرخان با CSS خالص (بدون جاوااسکریپت) انیمیشن داره، که باعث می‌شه عملکردش سبک و سریع باشه.
- **ریسپانسیو:** تو همه اندازه‌های صفحه (دسکتاپ، تبلت، موبایل) وسط صفحه می‌مونه و درست نمایش داده می‌شه.
- **UX:** متن "در حال بارگذاری..." به کاربر اطمینان می‌ده که سایت داره کار می‌کنه.
- **همخونی با تم:** رنگ آبی (#0078D4) با تم کلی سایت (که تو پاسخ‌های قبلی گفتم دارک با هایلایت آبیه) هماهنگه و حس یکپارچگی به کاربر می‌ده.

#### **چرا این طراحی رو انتخاب کردم؟**
- **سادگی:** دایره چرخان و متن ساده باعث می‌شه کاربر گیج نشه و فقط منتظر بمونه.
- **جذابیت:** تم آبی و انیمیشن نرم حس مدرن بودن به سایت می‌ده.
- **عملکرد:** استفاده از CSS برای انیمیشن باعث می‌شه لودینگ روی سرعت سایت تأثیر منفی نذاره.
- **کاربری:** نمایش لودینگ فقط تو زمان‌های لازم باعث می‌شه کاربر اذیت نشه.
