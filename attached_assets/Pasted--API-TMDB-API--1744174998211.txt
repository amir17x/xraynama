برای مستندسازی کامل و دقیق از API TMDB، به‌طور خاص برای استفاده بهینه از این API، در اینجا یک راهنمای کامل با تمام بخش‌های اصلی API و نحوه استفاده از آن آمده است که هوش مصنوعی می‌تواند بر اساس آن به بهترین شکل از آن استفاده کند:

---

### **مستندسازی کامل API TMDB**

#### **1. معرفی API TMDB**

API TMDB (The Movie Database) یک API قدرتمند است که برای دسترسی به اطلاعات مرتبط با فیلم‌ها، برنامه‌های تلویزیونی، افراد (بازیگران و کارگردانان)، و سایر داده‌های مربوط به دنیای سینما طراحی شده است. این API اطلاعاتی مانند جزئیات فیلم‌ها، نقدها، رده‌بندی‌ها، تصاویر، و بسیاری از داده‌های دیگر را فراهم می‌کند.

#### **2. دسترسی به API TMDB**
برای دسترسی به API TMDB، شما به یک **API key** نیاز دارید که می‌توانید آن را از وب‌سایت TMDB دریافت کنید. پس از دریافت API key، می‌توانید از آن برای ارسال درخواست‌ها به API استفاده کنید.

#### **3. ساختار URL درخواست‌ها**
آدرس‌های URL برای استفاده از API به شکل زیر هستند:

```bash
https://api.themoviedb.org/3/{نوع داده}/{شناسه} 
```

در اینجا `{نوع داده}` می‌تواند یکی از انواع زیر باشد:
- `movie` برای فیلم‌ها
- `tv` برای سریال‌ها
- `person` برای افراد
- `search` برای جستجو

و `{شناسه}` شناسه مورد نظر شما از هر نوع داده است که به آن اشاره خواهید کرد.

#### **4. نحوه استفاده از API**

##### **جستجو (Search)**
شما می‌توانید برای جستجو بر اساس نام فیلم، سریال یا شخص از متد جستجو استفاده کنید. این متد به شما امکان می‌دهد تا نزدیک‌ترین تطابق‌ها را برای جستجوی خود پیدا کنید.

**مثال درخواست جستجو برای فیلم "Jack Reacher":**

```bash
curl --request GET \
     --url 'https://api.themoviedb.org/3/search/movie?query=Jack+Reacher' \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json'
```

در این درخواست:
- `query=Jack+Reacher`: جستجو برای فیلم‌هایی که نامشان با "Jack Reacher" تطابق دارد.

##### **دریافت جزئیات یک رسانه خاص**
پس از دریافت نتایج جستجو، می‌توانید با استفاده از شناسه (`id`) هر فیلم، سریال یا شخص، درخواست جزئیات دقیق‌تری از آن را ارسال کنید.

**مثال درخواست برای دریافت جزئیات یک فیلم خاص با شناسه 343611:**

```bash
curl --request GET \
     --url 'https://api.themoviedb.org/3/movie/343611' \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json'
```

این درخواست، جزئیات کامل فیلم با شناسه 343611 را به شما می‌دهد.

##### **اضافه کردن داده‌های تکمیلی (Append to Response)**
اگر می‌خواهید داده‌های بیشتری از یک رسانه (مثلاً ویدیوها یا تصاویر) دریافت کنید، می‌توانید از ویژگی `append_to_response` استفاده کنید.

**مثال درخواست برای دریافت ویدیوهای یک فیلم:**

```bash
curl --request GET \
     --url 'https://api.themoviedb.org/3/movie/343611?append_to_response=videos' \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json'
```

##### **ردیابی تغییرات (Change Tracking)**
اگر می‌خواهید تغییرات به‌روز یک فیلم، سریال یا شخص را دنبال کنید، می‌توانید از روش‌های مختلف برای دریافت تغییرات استفاده کنید.

**مثال درخواست برای دریافت تغییرات یک فیلم خاص:**

```bash
curl --request GET \
     --url 'https://api.themoviedb.org/3/movie/343611/changes' \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json'
```

#### **5. خطاها و وضعیت‌ها**
در هنگام کار با API، ممکن است با کدهای مختلف وضعیت مواجه شوید. در اینجا برخی از مهم‌ترین کدهای وضعیت آورده شده است:

| کد | پیام خطا | توضیحات |
|----|----------|---------|
| 200 | موفقیت‌آمیز | درخواست با موفقیت انجام شده است. |
| 401 | عدم احراز هویت | API key معتبر نیست یا فراموش شده است. |
| 404 | داده پیدا نشد | شناسه یا مسیر درخواست شده موجود نیست. |
| 429 | محدودیت درخواست | تعداد درخواست‌ها بیش از حد مجاز است. |
| 500 | خطای داخلی سرور | مشکلی در سرور TMDB به وجود آمده است. |

#### **6. نحوه تنظیم محدودیت‌ها (Rate Limiting)**
API TMDB برای جلوگیری از استفاده زیاد از منابع، محدودیت‌هایی در تعداد درخواست‌ها دارد. از تاریخ 16 دسامبر 2019، محدودیت قبلی (40 درخواست در هر 10 ثانیه) برداشته شده است، اما هنوز محدودیت‌های دیگری در حدود 50 درخواست در هر ثانیه وجود دارد.

**توصیه:** اگر با خطای `429` مواجه شدید، این بدان معناست که تعداد درخواست‌ها بیش از حد مجاز است. بهتر است در این صورت درخواست‌های خود را با فاصله زمانی ارسال کنید.

#### **7. پشتیبانی از چند زبان**
TMDB به شما این امکان را می‌دهد که درخواست‌ها را به زبان‌های مختلف ارسال کنید. برای این کار، کافیست در انتهای URL پارامتر `language` را اضافه کنید.

**مثال درخواست برای دریافت جزئیات فیلم به زبان پرتغالی (pt-BR):**

```bash
curl --request GET \
     --url 'https://api.themoviedb.org/3/movie/343611?language=pt-BR' \
     --header 'Authorization: Bearer YOUR_API_KEY' \
     --header 'accept: application/json'
```

#### **8. مستندات رسمی**
برای اطلاعات بیشتر و آشنایی با دیگر ویژگی‌ها و پارامترها، می‌توانید مستندات رسمی API TMDB را در [وب‌سایت TMDB](https://www.themoviedb.org/documentation/api) مشاهده کنید.

---

این مستندات به شما کمک می‌کند که با API TMDB به‌طور کامل آشنا شوید و بتوانید درخواست‌های مختلفی را برای دریافت اطلاعات مربوط به فیلم‌ها، سریال‌ها و افراد به‌طور مؤثر ارسال کنید.
